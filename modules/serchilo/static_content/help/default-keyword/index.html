Setting up a default keyword will make all your [site:name] queries work. If a query is not recognized as a [site:name] command, the default keyword will be used.

Setting up a default keyword

without a user account Call [site:name] with default_keyword=[keyword],
e.g. [site:url]?default_keyword=g
with a user account Log in and go to My account -> Default keyword
