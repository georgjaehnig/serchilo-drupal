Make [site:name] the default search

  1. Go to your [site:name] homepage.

  2. Right click in the address bar and select Edit search engines.

  3. In Other search engines, find [site:name].

    It may be that [site:name] is missing. In this case, add it there:

    • Name: [site:name]
    • Keyword: [any you like, for instance s]
    • URL: [Homepage URL]?query=%s â€“ for instance: [site:url]n/en.usa?query=%s

    Point your mouse on that (added) line. A button Make default will appear. Click on it.

  4. Done. Now you can type [site:name] queries directly into your address field.
