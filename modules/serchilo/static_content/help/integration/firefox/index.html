Integration with Opensearch plugin

From Firefox 23 on, you can automatically use [site:name] in the address bar if you have the [site:name] Opensearch plugin installed and selected.

  1. Go to the [site:name] homepage

  2. In the OpenSearch input field, click on the Google logo:

  3. Click on Add [site:name] ...

  4. Done. Now you can enter all commands in this field and in the address bar.

To switch between the installed OpenSearch plugins

  1. click in the OpenSearch plugin field on the search engine icon icon
  2. choose another search engine. (If you don't find [site:name], restart Firefox.)

Hint: While browsing, put the focus directly into the OpenSearch textfield with Ctrl-K, on a Mac with Cmd-K.
