Install the Opensearch plugin

  1. Go on the [site:name] website.
  2. Check if language and country are correct, if not, change them.
  3. In the footer, right to Add to Browser, click on (Now)
  4. Confirm with Add.

    5. Make [site:name] the default search

    To use [site:name] as the default search in the Firefox address bar:

    1. Click on your mobile'sMenu button.
    2. Select Settings.
    3. Select Customize.
    4. Select Search settings.
    5. From the search engines list, select [site:name]. (You may need to scroll down)
    6. Click on Set as default.
