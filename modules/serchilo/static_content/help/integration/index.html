It is easy to integrate [site:name] directly into your browser or device. Many browser allow you to define a template URL which will be called when you enter a query.

examples [site:url]n/de.deu?query=%s
[site:url]u/jorges?query=%s
[site:url]n/de.deu?query=%s&default_keyword=g
templates [Homepage URL]?query=%s
[Homepage URL]?query=%s&default_keyword=[keyword]

The template URL consists of the homepage URL plus the query parameter. It defines which namespaces you want to use. If you haven't yet, you might want to read more about using namespaces.

The %s is the placeholder â€“ it will be replaced with your [site:name] queries.

You can also add a default keyword.
