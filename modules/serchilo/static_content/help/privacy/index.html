Calling a shortcut, without browsing the website

The following data gets recorded when calling a shortcut and directly being redirected to the target URL – for instance when using [site:name] via a browser plugin:

  • the shortcut id (as in /shortcuts/1/google-com)
  • the namespace id (an internal pointer to the shortcut's namespace)
  • the page type (console, url, api)
  • the source type (e.g. firefix-addon, opensearch, example)
  • whether the default keyword was used
  • the access time
  • the execution time

This data is used to collect statistics about shortcut usage, in order to improve their ranking in the autocomplete function and on the shortcuts overview.
